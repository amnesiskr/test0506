# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

이 저장소는 새로 시작하는 프로젝트입니다 ("첫 프로젝트"). 아직 언어, 프레임워크, 빌드 도구가 결정되지 않았습니다.

## 참고 사항

- 모든 답변은 한국어로 제공합니다.
- 코드 주석(인라인, 블록, docstring 등)은 반드시 한글로 작성합니다.

## 시작하기

프로젝트가 발전함에 따라 아래 내용을 이 파일에 추가해 주세요:
- 빌드 및 실행 명령어
- 테스트 실행 방법 (단일 테스트 포함)
- 린트/포맷 명령어
- 아키텍처 결정 사항 및 전체 구조
